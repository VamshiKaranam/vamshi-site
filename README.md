
# Vamshi Karanam - Personal Site (Next.js + Tailwind)

This is a minimal Next.js project (pages router) preloaded with a homepage component. It is ready to run locally and deploy to Vercel.

## Quick start (local)
1. Install dependencies
   ```bash
   npm install
   ```
2. Run dev server
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000

## Deploy to Vercel
1. Create a GitHub repo and push this project
2. Import the repo on https://vercel.com and deploy (Vercel auto-detects Next.js)

## Replace placeholders
- `public/cv.pdf` - add your real CV
- `public/headshot.jpg` - add your headshot
- Edit `pages/index.jsx` or `styles/globals.css` to customize text and styles
